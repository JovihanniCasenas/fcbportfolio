git bash commands:
cd [insert directory here, remove brackets] -> change directory *no need to use quotation marks
pwd -> to know present working directory
ls -> lists all contents inside the pwd
mkdir [insert name of new folder] -> makes new folder inside pwd
touch [insert name of new file with proper file format] -> creates new file inside pwd
git init -> creates new repo with the pwd
